# Tacky Dashboard

Static dashboard for Tacky. It is designed for GitHub Pages and stores editable bot/server settings in Supabase after login.

## Publish

Enable GitHub Pages for this repository. On first open, paste the Supabase project URL and anon public key, then sign in with the Supabase Auth email/password account.

## Secrets

Do not put Discord bot tokens, Supabase service role keys, passwords, or private credentials in this repository. The Discord token belongs only in the hosted bot process environment.
